# KRYKARD Image Generator

Generate and edit AI images using Bria API + Streamlit.

## Features
- HD image generation from prompt
- Custom style & aspect ratios
- Editable images (grayscale, sepia, blur, contrast)
- Background image UI
- Streamlit powered
## Python Version:
- recommended version python 10.0.x or python 11.0.x

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
